# Base normativa e técnica consultada

1. Regulation (EU) 2024/1781 — ESPR, especialmente artigos 9, 10 e 11:
   https://eur-lex.europa.eu/eli/reg/2024/1781/oj

2. Comissão Europeia — Digital Product Passport:
   https://single-market-economy.ec.europa.eu/single-market/digital-product-passport_en

3. Comissão Europeia — DPP Registry:
   https://single-market-economy.ec.europa.eu/single-market/digital-product-passport/dpp-registry_en

4. GS1 Digital Link:
   https://ref.gs1.org/standards/digital-link/

5. GS1-Conformant Resolver:
   https://ref.gs1.org/standards/resolver/

6. W3C Verifiable Credentials Data Model 2.0:
   https://www.w3.org/TR/vc-data-model/

## Observação
Os requisitos específicos para têxteis dependerão do ato delegado aplicável ao grupo de produtos. O pacote evita tratar campos setoriais ainda não confirmados como obrigação definitiva.
