# Governança de dados e modelos PI5

- somente feedback com `labelStatus=validated` entra no treino;
- toda validação exige responsável identificado;
- correções são novos eventos, nunca alteração silenciosa;
- versão do modelo e metodologia acompanha cada predição;
- promoção automática exige ao menos 70 exemplos validados;
- nenhuma promoção ocorre sem comparação champion/challenger;
- relatório por categoria evita melhora média com piora em subgrupos;
- dados insuficientes permanecem como estimativa experimental;
- snapshots e evidências devem permanecer auditáveis.
