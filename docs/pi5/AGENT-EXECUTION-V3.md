# PHYLLOS PI5 — Execução dos agentes v3

Esta entrega implementa os pacotes WP-01 a WP-04:

- geração determinística de fixtures sintéticos;
- segregação explícita de registros sintéticos;
- materialização de treino, validação, teste e quarentena por linhagem;
- baseline de regressão multissaída para score PI5;
- classificador de categoria para validar Top-1, Top-3 e matriz de confusão;
- calibração de confiança via conjunto de validação;
- MAE, RMSE, R², Pearson, Brier Score, ECE, intervalos preditivos e bootstrap;
- relatório sintético com advertência de uso;
- registro opcional do dataset e execução no PostgreSQL.

Os resultados sintéticos validam a esteira de software e não autorizam afirmações sobre desempenho real.
