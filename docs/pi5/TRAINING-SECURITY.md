# Segurança e Confidencialidade — Base PI5

- manter o repositório e os datasets em ambiente privado;
- usar credenciais separadas para desenvolvimento, revisão e produção;
- nunca armazenar chaves no código ou no dataset;
- limitar a API a `127.0.0.1` ou rede privada;
- criptografar backups;
- aplicar menor privilégio no PostgreSQL;
- separar informações pessoais dos registros de treinamento;
- registrar consentimento e retenção das imagens;
- manter o segredo de split fora do banco e do manifesto público;
- não publicar exemplos reais antes da decisão sobre patente e consentimentos.
