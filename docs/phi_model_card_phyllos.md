# Model Card PHYLLOS — PI5/Phi-5

## Estado
Experimental.

## Usos permitidos
- execução privada de testes;
- shadow mode;
- apoio a revisão humana;
- geração de hipóteses explicitamente marcadas;
- solicitação de evidências adicionais.

## Usos bloqueados
- afirmações técnicas, regulatórias ou socioambientais automáticas;
- confirmação visual de composição química;
- alteração automática de registros oficiais;
- promoção de hipótese para afirmação validada;
- decisão operacional de alto risco sem revisão;
- divulgação de percentuais sintéticos como desempenho real.

## Evidência disponível
Apenas smoke test sintético da infraestrutura.

## Limitações
- ausência de avaliação física;
- configuração Phi-5 incompleta;
- classificação de categorias insuficiente;
- abstinência, robustez, segurança e operação ainda não mensuradas.

## Governança
Toda saída deve preservar:
- versão do modelo e prompt;
- evidências utilizadas;
- estado epistemológico;
- possibilidade de indeterminação;
- trilha de auditoria;
- mecanismo de revisão humana.
