# PHYLLOS — Onboarding e Central de Ajuda

A entrega cria um módulo permanente `Início e ajuda`, aberto automaticamente no primeiro acesso.

## Recursos

- primeiro acesso guiado;
- links rápidos orientados por tarefas;
- rotas recomendadas conforme o objetivo da usuária;
- mapa de comunicação entre os módulos;
- explicação de entradas e saídas de cada módulo;
- checklist de adoção calculado a partir dos dados locais e da API PI5;
- consultas rápidas pesquisáveis;
- glossário em linguagem simples;
- botão flutuante de ajuda;
- navegação e layout responsivos.

## Persistência local

- `phyllos-onboarding-seen-v1`: registra que o guia foi apresentado;
- `phyllos-onboarding-progress-v1`: registra o último progresso calculado.
